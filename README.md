# Smart Web Design Scraper

Smart Web Design Scraper

## How to Run

### Angular Client
```
cd client
npm start
```
It will start angular dev server at port 3301

### Test Site
```
cd test-site
docker-compose up
```
It will start a sample site for testing purpose at port 3303
