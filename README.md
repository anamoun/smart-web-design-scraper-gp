# Smart Web Design Scraper

Smart Web Design Scraper

## Running development mode

```sh
# Terminal 1: Express
cd server
npm install
npm start

# Terminal 2: Angular
cd client
npm install
npm start

# Terminal 3: Mongo
docker-compose -f mongo.docker-compose.yml up
```
## Running the app

```sh
cd client
npm install
cd ..
cd server
npm install
cd ..
docker-compose up
```
